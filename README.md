# Что это?
Плагин для автоматизации работы с МЭШ обладающий следующим функционалом:
- выставление "Без ДЗ" на все даты (даже те которые были заблокированы)
- синхронизация КТП (автоматически достраивает и пересчитывает все ваши КТПшки)
- Выставление оценок (выставляет оценки по среднему баллу студента, если балл отсутствует выставляет **3**)
